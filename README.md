# Ishii-Morphine-2025

Analysis code for **Ishii et al., 2025** — whole-brain c-Fos mapping of the opioid/morphine
response. This repository is **code-only**; all raw and processed data live in the accompanying
**Figshare deposit** (see *Data* below).

## Structure

```
analysis/
  0X_<group>/                  one folder per data source
    notebooks/                 cleaned marimo (.py) notebooks + *_CHANGES.md notes
    data/  figures/  tables/   placeholders — resolved from the Figshare deposit at run time
  shared/                      atlas / helpers shared across groups
  MAPPING.md                   manuscript figure/table -> notebook map
  DATA_MANIFEST.md             per-group list of required inputs (in deposit / provide / regen)
requirements.txt
```

Notebooks are [marimo](https://marimo.io) notebooks (`.py`). Open one with
`marimo edit analysis/01_main_cfos_morphine/notebooks/Fig1_S2_whole_brain_cFos.py`.

## Data

Raw + processed data are in the Figshare deposit, laid out to mirror `analysis/` (per-group
folders + `shared/atlas`). Notebooks resolve data via an environment variable:

```bash
export OPIOID_DATA_ROOT=/path/to/Figshare_deposit
```

`analysis/DATA_MANIFEST.md` lists, per group, which inputs ship in the deposit, which are large
files to copy, and which are regenerated by running an earlier notebook.
